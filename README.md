# Wedding site folder

This is a ready-to-run Vite + React + Tailwind project with the mandala image already wired in as a fixed background layer.

## What is already done

- The mandala PNG is included in `src/assets/`
- The background is fixed and stays still while the page scrolls
- The white overlays and card opacity were adjusted so the mandala actually shows through
- Your RSVP endpoint from the uploaded script is still in place

## How to run locally

1. Open the folder in VS Code or CodeSandbox
2. Run:

```bash
npm install
npm run dev
```

3. Open the local URL shown in the terminal

## Main files

- `src/App.jsx` — main website component
- `src/assets/mandala_mirrored_full_zoomed_out_with_final_dots.png` — background image
- `src/index.css` — Tailwind imports and base styles

## If using CodeSandbox

Create a new Vite React sandbox, then upload and replace the project files with this folder.
